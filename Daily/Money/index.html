    四、学习路径建议（按阶段）

    🔰 入门（0–6 个月）

    • 先看：The Plain Bagel、李永乐、両学長、半佛
    • 目标：搞懂 股票 / 基金 / ETF / 指数 / 定投 / 风险 / 复利，建立正确观念

    📈 进阶（6–18 个月）

    • 指数 / 配置：Ben Felix、有知有行、Whiteboard Finance
    • 价值 / 估值：The Swedish Investor、Sven Carlin、Aswath Damodaran
    • 技术分析：Rayner Teo

    🏛 高阶（18 个月 +）

    • 宏观 / 机构：Real Vision、CNBC、Damodaran
    • 个股 / 行业：Jeremy、Warren 王、许桐锴、バフェット太郎

    五、重要提醒（避坑）

    1. 拒绝 “荐股 / 喊单 / 短期暴富”：凡是天天给代码、保证收益、短线暴利的，一律拉黑。
    2. 区分 “教学” 与 “营销”：很多频道卖课程 / 社群，先看免费内容质量，再决定付费。
    3. 独立思考：YouTube 是学习工具，不是决策依据。所有观点都要自己验证。
    4. 先小资金实践：看懂≠会做。先用小钱练手，建立自己的体系。

    Episode 1: The Fundamentals – Why is investing important?

    约 637 个字 预计阅读时间 2 分钟

    内容

    这一集是整个系列的开篇，不讲具体产品，只讲为什么普通人必须投资。

    • 解释通货膨胀如何持续侵蚀现金的购买力，钱放着不动会越来越不值钱。
    • 对比存钱 vs 投资：储蓄只能保值一小段时间，长期必然跑输通胀。
    • 说明投资的核心目的：让资产增速超过通胀，实现财富真正增长。
    • 简单提到复利的早期概念，强调时间是投资最大的优势。
    • 点明投资不是富人专属，普通人靠长期、简单、低成本投资也能实现财务安全。

    重点

    1. 通胀是现金的最大敌人，持有现金长期必然贬值。
    2. 投资的根本目的是对抗通胀 + 实现财富增值，而不是投机暴富。
    3. 复利 + 时间是普通人最可靠的财富增长方式。
    4. 投资不是赌博，而是让钱为你工作，替代单一靠劳动收入。
    5. 建立基础观念：不投资的风险，其实比投资更大。

    Episode 2: Bonds – Using Debt to Invest

    内容

    讲解债券的本质：投资者借钱给政府、企业等机构，换取定期利息与到期本金偿还。解释债券如何作为 “债务工具” 参与投资，介绍票息、到期日、发行人信用、价格与利率的反向关系等基础概念。

    重点

    • 债券 = 债务凭证，你是债权人
    • 收益主要来自固定利息
    • 价格与市场利率呈反向变动
    • 风险低于股票，更偏稳健
