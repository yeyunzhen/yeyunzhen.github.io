    效果器

    约 3190 个字 预计阅读时间 11 分钟

    参考某乐手的效果器板进行学习。感谢他的分享。

    效果器案例

    效果器名称 效果类型 具体说明（参数含义 + 调节方法）
    Providence VELVET COMP 光学压缩（Optical Compressor） - SUSTAIN（延音 / 压缩强度）：控制压缩强度与延音长度。

    → 逆时针：压缩弱、延音短，限幅器模式；顺时针：压缩强、延音长，平滑丝绒感。

    - ATTACK（启动时间）：控制音头响应速度。

    → 逆时针（快）：压制音头、更紧实；顺时针（慢）：保留音头、动态自然。

    - LEVEL（输出音量）：补偿压缩音量损失，匹配干声输出。
    Fulltone OCD 过载 / 失真（Overdrive/Distortion） - DRIVE（增益 / 失真度）：控制失真强度，从轻微推弦过载到高增益失真。

    → 9 点以下：轻过载；9–12 点：经典中度过载；12–3 点：高增益失真。

    - TONE（音色）：主要控制高频，12 点为中性。

    → 逆时针：削减高频、音色更暖；顺时针：提升高频、更亮更脆。

    - VOLUME（音量）：控制总输出，可作推子使用。

    → 低 Drive 时 10–12 点为均衡音量；高 Drive 时适当降低。

    - HP/LP 开关：

    → HP：中频突出、增益更高，适合双线圈 / 摇滚 / 金属。

    → LP：透明低染色、动态最大化，适合单线圈 / 轻过载 / 推箱。
    VEMURAM Jan Ray 经典过载（Vintage Overdrive） - GAIN（增益）：控制过载深度，范围从清音激励到中度失真。

    → 顺时针增大，失真更明显、延音更长。

    - TONE（音色）：被动式音色控制，调节高频与整体通透度。

    → 12 点中性；顺时针更亮、穿透力强；逆时针更暖、平滑。

    - VOLUME（音量）：输出电平控制，用于匹配音箱音量或作激励。

    → 低 Gain 时拉满可作 Clean Boost；高 Gain 时适当降低。

    - 特色：基于经典 Dumble 音箱电路，动态极佳、音色温暖通透，单 / 双线圈适配。
    Walrus Audio JULIA 合唱 / 颤音（Chorus/Vibrato） - RATE（速率）：调制速度，控制合唱 / 颤音的快慢。

    → 顺时针加快，效果更活跃；逆时针变慢，更舒缓。

    - DEPTH（深度）：调制幅度，控制效果强度。

    → 顺时针加深，效果更明显；逆时针变浅，更 subtle。

    - MIX（干湿比）：效果信号与干声的混合比例。

    → 0%：仅干声；50%：均衡混合；100%：仅效果声。

    - VIBE/CHORUS 开关：

    → VIBE：纯颤音（音高波动），无干声混合。

    → CHORUS：经典合唱（多声部叠加），保留干声。

    - WIDTH（宽度）：立体声宽度，控制左右声道分离度。
    strymon TIMELINE 多功能数字延迟（Digital Delay） - TIME（延时时间）：原始声与第一个回声的间隔（0–3000ms）。

    → 配合 Tap Tempo 可设置节奏时值。

    - FEEDBACK（回授）：控制回声重复次数与衰减速度。

    → 0%：无重复；100%：接近无限重复。

    - MIX（干湿比）：延时信号音量比例。

    - TYPE（延迟类型）：12 种模式（Digital、Analog、Tape、Dual、Reverse 等）。

    - PARAM 1/2：自定义参数（如调制速度、滤波、预延迟等，随模式变化）。

    - LOOPER：内置循环录音，最长 120 秒。
    strymon blueSky 多功能混响（Reverb） - MIX（干湿比）：混响信号与干声比例。

    - DECAY（衰减时间）：混响尾音持续长度。

    → 顺时针更长（大厅 / 空间）；逆时针更短（房间 / 板式）。

    - PRE-DELAY（预延迟）：干声与混响 onset 之间的间隔，增加清晰度。

    - DIFFUSION（扩散度）：控制混响密度，越密越像空间，越疏越像回声。

    - TYPE（混响类型）：3 种核心模式（ROOM、HALL、PLATE）+ MOD（调制）。

    - DAMPING（阻尼）：高频衰减控制，顺时针更暗、更暖；逆时针更亮、更脆。
    Keeley Loomer 混响 + 延迟（Reverb + Delay） - DELAY TIME：延迟时间（0–1000ms）。

    - FEEDBACK：延迟重复次数。

    - MIX：延迟 + 混响总干湿比。

    - REVERB：混响强度与衰减时间。

    - TONE：整体音色（高频）控制。

    - MODE 开关：

    → DELAY：纯延迟模式。

    → REVERB：纯混响模式。

    → COMBINE：延迟 + 混响并行，营造深度空间感。

    - 特色：紧凑双效果，适合营造氛围与空间感。

    压缩技巧

    • Attack（启动）：管音头 快 = 压音头，声音更紧 慢 = 留音头，更有颗粒感
    • Release（释放）：管延音 快 = 音符干净、不糊 慢 = 延音长、更平滑
    • Sustain / Level：管音量平稳度 + 延音长短
    • Level：补偿音量，让压缩后声音不变小

    1. 轻压缩：让音色更 “稳”、更专业

    • 用途：节奏、清音
    • 思路：只压一点点，让强弱不均的弹奏变均匀

    • 设置：

      • Sustain 小
      • Attack 偏慢（保留音头）
      • Release 中等

    • 听感：更顺滑、更专业，不刺耳、不乱

    2. 强压缩：增加延音

    • 用途：后摇、氛围、清音延长、主奏 solo
    • 思路：把声音 “拉长”，提升延音

    • 设置：

      • Sustain 大
      • Attack 中等
      • Release 慢

    • 听感：像开了无限延音，空灵、大气

    3. 快 Attack + 快 Release：让声音 “紧实、有力度”

    • 用途：funk、放克、切音、节奏
    • 思路：压掉多余音头，让音符短促有力

    • 设置：

      • Attack 快
      • Release 快
      • Sustain 中等

    • 听感：颗粒清晰、不拖泥带水、非常 “脆”

    4. 慢 Attack：保留拨弦质感，动态最好

    • 用途：布鲁斯、乡村、原声、指弹
    • 思路：不压音头，只压后面的延音

    • 设置：

      • Attack 慢
      • Release 中等

    • 听感：有弹性、动态自然、像好音箱的音色

    5. 压缩当 “激励 / 推子” 用

    • 用途：推音箱前级、提升清音饱满度
    • 思路：轻微压缩 + 提升音量

    • 设置：

      • Sustain 很小
      • Level 开大

    • 听感：声音更饱满、更紧，但不过载

    过载技巧

    • Drive / Gain（失真度）：管沙哑、饱满、失真强度 小 = 更干净、更像美化 大 = 更冲、更碎、更失真
    • Tone（音色）：管明暗、亮度、穿透力 小 = 暗、暖、柔 大 = 亮、脆、尖
    • Level / Volume（音量）：管输出大小、推箱力度 可用来平衡音量，也可以当激励推子

    1. 轻过载：让音色更 “润”、更专业

    • 用途：清音节奏、分解
    • 思路：几乎听不出失真，只让声音变厚、变顺滑

    • 设置：

      • Drive 很小
      • Tone 中性偏暖
      • Level 与干声持平

    • 听感：不干不刺、更饱满、更像好音箱的音色

    2. 标准过载：节奏、solo 全能

    • 用途：流行、摇滚、英伦、布鲁斯
    • 思路：有明显颗粒感，但动态依然很好

    • 设置：

      • Drive 中等
      • Tone 12 点左右
      • Level 适中

    • 听感：清晰、有弹性、轻弹干净重弹冲

    3. 偏重过载：硬摇滚 Riff、强力和弦

    • 用途：硬摇、另类摇滚、垃圾摇滚
    • 思路：失真更明显，中频突出、力度强

    • 设置：

      • Drive 偏大
      • Tone 略暗一点避免糊
      • Level 适当

    • 听感：紧实、有力、riff 清晰不散

    4. 当 Clean Boost 用：只加劲儿不加失真

    • 用途：推音箱、提升清音厚度、solo 切音量
    • 思路：Drive 最小，只靠 Level 提升音量

    • 设置：

      • Drive 几乎关死
      • Tone 中性
      • Level 开大

    • 听感：声音更饱满、更紧，但依然是清音

    5. 推高增益失真：让 solo 更猛更透

    • 用途：金属 solo、主奏音色
    • 思路：用过载推失真前级，增加中频与力度
    • 听感：更紧、更冲、穿透力更强

    6. 控制 Touch Sensitivity

    • 用途：布鲁斯、爵士、抒情 solo
    • 思路：7点钟Vulumn。Drive 调到临界点，靠手指力度控制动态
    • 听感：轻弹柔和，重弹有冲击力，非常 “唱歌性”

    延时技巧

    • Time / Delay Time（时间）：管回声间隔快慢 小 = 紧密、Slapback、快速节奏 大 = 空旷、悠远、氛围
    • Feedback（重复次数）：管回声重复多少 小 = 只响一两声，干净 大 = 反复回荡，空间感强
    • Mix / Level（混合 / 音量）：管延时声音大小 小 = 若隐若现，点缀 大 = 明显突出，氛围感拉满
    • Tone / Damping（音色 / 高频衰减）：管回声明暗 亮 = 清晰、清脆 暗 = 柔和、朦胧、复古

    1. 短延时 Slapback：复古、乡村、摇滚

    • 用途：乡村、老式摇滚、人声贴脸感
    • 思路：极短时间、极少重复，像一次反射回声

    • 设置：

      • Time 极短：60–150ms
      • Feedback 很小
      • Mix 适中

    • 听感：声音变厚、有弹性、复古紧实

    2. 1/4 拍节奏延时：抒情、流行

    • 用途：流行、抒情、弹唱、主奏 solo
    • 思路：跟拍点对齐，清晰不糊

    • 设置：

      • Time 对齐歌曲速度（Tap Tempo）
      • Feedback 中等
      • Mix 偏小点缀

    • 听感：空间感自然、更饱满

    3. 长延时氛围：后摇、梦幻、空灵

    • 用途：后摇、氛围、间奏、留白段落
    • 思路：时间拉长、重复多，形成音墙

    • 设置：

      • Time 较长：500ms–2s
      • Feedback 偏高
      • Mix 偏大

    • 听感：辽阔、梦幻、延音拉长

    4. 低混低重复：悄悄加宽声音（最实用）

    • 用途：节奏、打底
    • 思路：几乎听不出延时，只让声音更立体

    • 设置：

      • Time 中等
      • Feedback 很小
      • Mix 很低

    • 听感：更宽、更润、不抢戏、不糊

    5. 暗音色延时：柔和、高级、不刺耳

    • 用途：抒情、慢歌、清音分解
    • 思路：把延时音色调暗，避免高频刺耳

    • 设置：

      • Tone/Damping 偏暗
      • Time 中等
      • Mix 适中

    • 听感：顺滑、温柔、高级感强

    6. 乒乓 / 双延时：立体声更宽、更华丽

    • 用途：solo、间奏、铺底音色
    • 思路：左右来回跳，声场拉满

    • 设置：

      • 开启 Ping Pong
      • Time 中等
      • Mix 适中

    • 听感：包围感强、更现代、更专业

    混响技巧

    • Decay / Time（衰减时间）：管空间大小、尾音长短 小 = 小房间、紧凑、贴耳 大 = 大厅、教堂、空旷、悠远
    • Pre Delay（预延迟）：管干声和混响的间隔 小 = 混响紧贴人声，更融合 大 = 声音更清晰、更靠前、不糊
    • Mix / Level（干湿比）：管混响音量大小 小 = 轻微点缀，更专业 大 = 空间感强、氛围浓厚
    • Damping / High Cut（高频阻尼 / 低通）：管混响明暗 小 = 亮、脆、清晰 大 = 暗、柔、暖、朦胧

    1. 短混响（房间 / 板式）：贴耳、紧凑、专业

    • 用途：流行节奏、弹唱、清音、人声贴脸
    • 思路：时间短、不拖沓，只给一点点空间

    • 设置：

      • Decay 很短：0.5–1.2s
      • Pre Delay 中等
      • Mix 很小

    • 听感：干净、紧实、不飘、很 “录音室”

    2. 中等混响：抒情、solo、好听不腻

    • 用途：主奏 solo、抒情、分解和弦
    • 思路：自然空间感，不抢戏、提升质感

    • 设置：

      • Decay：1.2–2.0s
      • Pre Delay：10–20ms
      • Mix 适中

    • 听感：圆润、有呼吸感、高级不糊

    3. 长大厅 / 教堂混响：氛围、空灵、后摇

    • 用途：后摇、氛围、间奏、留白、梦幻音色
    • 思路：时间拉长，营造巨大空间

    • 设置：

      • Decay：2.5–4s+
      • Pre Delay 稍大
      • Mix 偏大

    • 听感：辽阔、空灵、仙气、沉浸感强

    4. 高阻尼暗混响：温暖、柔和、不刺耳

    • 用途：慢歌、清音、过载 solo、舒缓段落
    • 思路：切掉高频，混响更柔和不扎耳

    • 设置：

      • Damping / High Cut 偏大
      • Decay 中等
      • Mix 适中

    • 听感：温暖、朦胧、顺滑、高级

    5. 大 Pre Delay：声音清晰、靠前、不糊

    • 用途：弹唱、节奏
    • 思路：混响往后退，干声突出，不影响清晰度

    • 设置：

      • Pre Delay：20–40ms
      • Decay 中等
      • Mix 适中

    • 听感：靠前，空间感还有但不乱

    6. 极低混响：“隐形美化”，几乎听不出来

    • 用途：打底、节奏、录音标准用法
    • 思路： barely 听到混响，只让声音不干

    • 设置：

      • Decay 短
      • Mix 非常小

    • 听感：更润、更立体、更专业，却不觉得加了效果

    合唱技巧

    • Rate（速率）：管波动的快慢 小 = 慢、宽、温柔、空灵 大 = 快、活泼、抖动感强
    • Depth（深度）：管波动的幅度 / 强弱 小 = 很淡、只加宽声音、不明显 大 = 浓厚、漂浮感强、很 “合唱”
    • Mix / Level（混合 / 音量）：合唱效果的多少 小 = 轻微润色 大 = 效果明显、华丽

    1. 浅合唱：加宽声音，隐形美化

    • 用途：清音节奏、分解
    • 思路：几乎听不出 “晃”，只让声音更宽更厚

    • 设置：

      • Rate 偏慢
      • Depth 很小
      • Mix 偏低

    • 听感：更立体、更饱满、不干瘪，像合奏

    2. 标准合唱：清新、舒缓、流行音色

    • 用途：流行、轻摇滚、抒情、英伦、Clean 音色
    • 思路：明显但不夸张的合唱感

    • 设置：

      • Rate 中等
      • Depth 中等
      • Mix 适中

    • 听感：干净、梦幻、温柔、有空气感

    3. 慢 Rate + 大 Depth：大气、空灵、氛围

    • 用途：后摇、氛围、间奏、长音铺底
    • 思路：慢而深的波动，营造宽广漂浮感

    • 设置：

      • Rate 慢
      • Depth 偏大
      • Mix 适中

    • 听感：辽阔、仙气、梦幻、像在大空间里

    4. 快 Rate + 小 Depth：复古、80 年代、迷幻

    • 用途：复古摇滚、放克、冲浪摇滚、复古流行
    • 思路：快速细微抖动，经典复古味儿

    • 设置：

      • Rate 偏快
      • Depth 中等偏小
      • Mix 适中

    • 听感：活泼、跳跃、年代感强

    5. 合唱 + 过载：肥厚、温暖、有质感

    • 用途：轻过载节奏、布鲁斯、抒情 solo
    • 思路：用合唱让过载更厚更柔

    • 设置：

      • Rate 偏慢
      • Depth 中等
      • Mix 适中

    • 听感：温暖、肥厚、不刺耳、更有层次

    6. 合唱 + 延时 + 混响：极致空间氛围

    • 用途：后摇、氛围段落、长音 solo
    • 思路：合唱加宽 → 延时拉长 → 混响铺开
    • 听感：立体、宏大、梦幻、包围感极强
